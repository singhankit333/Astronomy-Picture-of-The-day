# Astronomy-Picture-of-The-day
APOD: Astronomy Picture of The day, is one of the NASA Open APIs where each day a unique image of the space is uploaded.
The webpage uses NASA API, APOD to fetch the image of space for that day.
The web page takes date as input using a datepicker.
After submitting the date, the Astronomy picture of the day for that date get displayed on the page. Also displayed is the title of the image and explanation of the image.
